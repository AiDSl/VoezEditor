Starting a new Project

In order to start a new project, the following files need to be placed into the ActiveProject folder:
  • The audio file for the song. This should be a .wav file whose name starts with "song_".
  • Optionally, a background image for the song. This should be a .png file whose name starts with "image_".
Note that the length of your audio file will be the length of your project's song. If you want your song to be shortened, you need to edit the file with an audio editor.
